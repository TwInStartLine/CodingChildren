# CodingChildren
Script to configure the study environment.

1) Just excute "setup.bat".
2) Check virtual drive name "S:" and initial folder as below.

        S:\
            +-- C           (folder)
            |
            +-- Python      (folder)
            |
            +-- Scratch     (folder)
            |
            +-- go.bat      (file)
4) [CAUTION!] Do not remove or rename "go.bat" file.
5) Let's start!!
